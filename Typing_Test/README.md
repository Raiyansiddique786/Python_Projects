# ⌨️ Typing Speed Test

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Project Type](https://img.shields.io/badge/Project-Mini%20Project-green)
![Status](https://img.shields.io/badge/Status-Completed-brightgreen)

A simple **Python console application** that measures your **typing speed and accuracy**.
The program displays a random sentence, records the time taken by the user to type it, and calculates **Words Per Minute (WPM)** and **typing accuracy**.

---

## 🚀 Features

* Random sentence generation
* Calculates **Words Per Minute (WPM)**
* Calculates **typing accuracy**
* Measures typing time
* Beginner-friendly Python project

---

## 🛠️ Technologies Used

* Python
* Built-in modules:

  * `time`
  * `random`

---

## ⚙️ How It Works

1. The program randomly selects a sentence.
2. The user presses **Enter** to start the test.
3. The user types the displayed sentence.
4. The program calculates:

   * ⏱ Time taken
   * ⚡ Words per minute (WPM)
   * 🎯 Accuracy percentage

---

## ▶️ How to Run the Project

1. Make sure **Python** is installed.

2. Download or clone this repository.

3. Run the script:

```bash
python typing_speed_test.py
```

4. Type the sentence shown on the screen and check your results.
